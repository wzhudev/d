# dotfiles

Efficient development environment for _practical minimalists_. 

* VSCode (VSCodeVim)
* Node.js, Rust, Go
* mise, fzf, lazygit, gh

Pick your operating system to get started:

- [macOS](./platforms/mac/README.md)
