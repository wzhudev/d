# dotfiles

An efficient development environment for _practical minimalists_. Read about [Tools](https://wzhu.dev/tools).

* Editors: VSCode (VSCodeVim)
* Languages: Node.js, Rust, Go
* Developer tools: mise, fzf, lazygit, gh
